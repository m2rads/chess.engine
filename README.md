# TinyChess 

Tiny chess engine for learning purposes


Built with fastapi and vanila react

to run: 

```bash

python -m venv .venv
source .venv/bin/activate

pip install "fastapi[standard]"
pip install -r requirements.txt

cd frontend 
npm install

cd ..

fastapi dev
```

Then navigate to 

http://127.0.0.1:8000/
